# Sector Five

Ruby video games using sprites.

## Setup

Sector Five uses four different types of sprites

  * Player's Spaceship
  * Enemy Ships
  * Bullets
  * Explosions

## Play

To start the game cd into the directory and run the following command

`$ ruby sector_five.rb`
